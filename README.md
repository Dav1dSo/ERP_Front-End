Clone the project

```bash
  git clone https://github.com/Dav1dSo/ERP_Front-End.git
```

Enter the project directory

```bash
  cd ERP_Front-End
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
